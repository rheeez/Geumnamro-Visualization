# 금남로 접근성 분석 대시보드

광주광역시 금남로 보행 접근성 분석 및 시각화 프로젝트

## 프로젝트 소개

이 프로젝트는 광주 금남로 일대의 보행 접근성을 측정하고 분석한 데이터를 시각화한 대시보드입니다. 유효폭, 단차, 기울기 등 장애물 없는 생활환경(Barrier Free) 인증기준에 따라 측정된 데이터를 기반으로 합니다.

### 새로운 기능: CSV 데이터 관리

**코드 수정 없이 데이터만 업데이트!**

- CSV 파일을 엑셀이나 메모장으로 수정
- 웹사이트를 새로고침하면 자동 업데이트
- 데이터 추가/삭제/수정 자유롭게
- [데이터 편집 가이드 보기](DATA_GUIDE.md)

### 주요 기능

- **법적 기준 표시**: 유효폭(1.2m), 기울기(4.8°), 단차(2cm) 기준 시각화
- **히트맵 분석**: 장소별 접근성 지표를 색상으로 표현
- **박스플롯 차트**: 각 측정값의 분포와 이상치 확인
- **인터랙티브 툴팁**: 호버 시 상세 정보 표시
- **반응형 디자인**: 데스크톱 및 모바일 환경 지원

## 시작하기

### 요구사항

- 모던 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 인터넷 연결 (외부 라이브러리 로드용)

### 설치 및 실행

1. 저장소 클론
```bash
git clone https://github.com/rheeez/Geumnamro-Visualizatoin.git
cd geumnamro-visualization
```

2. HTML 파일 열기
```bash
# 브라우저에서 index.html 파일을 직접 열거나
# 로컬 서버 실행 (Python 예시)
python -m http.server 8000
```

3. 브라우저에서 `http://localhost:8000` 접속

## 프로젝트 구조

```
geumnamro-analysis/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css       # 모든 스타일시트
├── js/
│   └── main.js         # 데이터 및 차트 로직
├── data/               # 📊 데이터 파일 (CSV)
│   ├── width_data.csv  # 유효폭 측정 데이터
│   ├── step_data.csv   # 단차 측정 데이터
│   └── slope_data.csv  # 기울기 측정 데이터
├── DATA_GUIDE.md       # 데이터 편집 가이드
└── README.md           # 프로젝트 문서
```

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 애니메이션
- **JavaScript (ES6+)**: 데이터 처리 및 차트 렌더링
- **Chart.js**: 통계 차트 라이브러리
- **Plotly.js**: 인터랙티브 박스플롯
- **Google Fonts**: Noto Sans KR 폰트

## 데이터 측정 기준

### 법적 기준 (장애물 없는 생활환경 인증기준)

| 항목 | 기준값 | 단위 |
|------|--------|------|
| 유효폭 | 1.2 | m |
| 기울기 | 4.8 | 도(°) |
| 단차 | 2.0 | cm |

### 측정 장소

금남로 일대 주요 상업시설 및 공공건물 포함:
- 은행 (우리은행, NH농협은행, 하나은행 등)
- 상업시설 (YMCA, 투썸플레이스, 노스페이스 등)
- 공공시설 (5·18민주화운동기록관 등)

## 시각화 요소

### 1. 히트맵
- 장소별 접근성 지표를 색상 강도로 표현
- 호버 시 상세 측정값 표시

### 2. 박스플롯
- 유효폭, 단차, 기울기의 데이터 분포
- 법적 기준선 표시
- 이상치 확인 가능

### 3. 상태 분류
- **양호**: 모든 기준 충족
- **개선(장기)**: 일부 기준 미달
- **긴급조치**: 즉각적인 개선 필요
