# 데이터 수정 가이드

## 개요
이 프로젝트는 데이터를 CSV 파일로 분리하여 관리합니다. 
**코드를 수정하지 않고** CSV 파일만 편집하면 차트가 자동으로 업데이트됩니다! 🎉

## 데이터 파일 위치

```
data/
├── width_data.csv    → 유효폭 데이터 (미터)
├── step_data.csv     → 단차 데이터 (센티미터)
└── slope_data.csv    → 기울기 데이터 (도)
```

## 🔧 편집 방법

### 방법 1: 엑셀로 편집

1. **CSV 파일 열기**
   - CSV 파일을 더블클릭하면 엑셀에서 열림
   - 또는 엑셀에서 `파일 > 열기` 선택

2. **데이터 수정**
   - 장소 이름, 측정값 수정
   - 행 추가/삭제 가능
   - **첫 번째 줄(헤더)은 절대 수정하지 마세요!**

3. **저장하기**
   - `파일 > 저장` (Ctrl+S)
   - **파일 형식을 CSV로 유지해야 합니다!**

### 방법 2: 메모장으로 편집

1. **CSV 파일 열기**
   - CSV 파일 우클릭 → `연결 프로그램` → `메모장`

2. **데이터 수정**
   ```csv
   장소,유효폭
   투썸플레이스,1.5
   금남로공원,3.6
   ```
   - 콤마(,)로 구분
   - 각 줄은 하나의 데이터

3. **저장하기**
   - `파일 > 저장` (Ctrl+S)

## CSV 파일 형식

### width_data.csv (유효폭)
```csv
장소,유효폭
투썸플레이스,1.5
금남로공원,3.6
YMCA,0.9
```

- **장소**: 측정 장소 이름 (한글, 영문 가능)
- **유효폭**: 미터 단위 (소수점 가능, 예: 1.5, 2.0)

### step_data.csv (단차)
```csv
장소,단차
YMCA,3.5
NH농협은행,18.0
루이까스텔,21.0
```

- **장소**: 측정 장소 이름
- **단차**: 센티미터 단위 (소수점 가능, 예: 3.5, 18.0)

### slope_data.csv (기울기)
```csv
장소,기울기
금남로공원,2.4
투썸플레이스,3.6
SC제일은행,9.7
```

- **장소**: 측정 장소 이름
- **기울기**: 각도(도) 단위 (소수점 가능, 예: 2.4, 9.7)

## 데이터 추가하기

엑셀이나 메모장에서 새로운 줄 추가:

**엑셀:**
```
마지막 행 아래에 새 행 추가 후 입력
```

**메모장:**
```csv
장소,유효폭
기존데이터,1.5
새장소,2.3    ← 이렇게 추가
```

## 데이터 삭제하기

**엑셀:**
```
삭제할 행 선택 → 우클릭 → 삭제
```

**메모장:**
```csv
장소,유효폭
투썸플레이스,1.5
삭제할데이터,3.6    ← 이 줄 전체 삭제
금남로공원,2.4
```

## 확인하기

1. **CSV 파일 저장**
2. **웹 브라우저에서 index.html 새로고침 (F5)**
3. **차트가 자동으로 업데이트됨!**

## 주의사항

### 꼭 지켜주세요!

1. **헤더는 절대 수정하지 마세요**
   ```csv
   장소,유효폭    ← 이 첫 줄은 그대로 유지!
   ```

2. **CSV 형식 유지**
   - 엑셀에서 저장할 때 "CSV (쉼표로 분리)" 형식으로 저장
   - 다른 형식(xlsx, xls)으로 저장하면 작동 안 함!

3. **콤마(,) 사용 금지**
   - 장소 이름에 콤마 사용 불가
   - 잘못된 예: `투썸플레이스,광주점,1.5`
   - 올바른 예: `투썸플레이스 광주점,1.5`

4. **숫자만 입력**
   - 유효폭/단차/기울기 열에는 숫자만 입력
   - 잘못된 예: `1.5m`, `약 2.0`
   - 올바른 예: `1.5`, `2.0`

5. **빈 줄 금지**
   - 데이터 사이에 빈 줄을 넣지 마세요

## 문제 해결

### "차트가 표시되지 않아요"

1. **F12를 눌러서 개발자 도구 열기**
2. **Console 탭에서 에러 확인**
3. **자주 발생하는 에러:**

#### "CSV 파일을 불러올 수 없습니다"
- data 폴더가 index.html과 같은 위치에 있는지 확인
- CSV 파일 이름이 정확한지 확인
  - `width_data.csv` (O)
  - `Width_Data.csv` (X) - 대소문자 구분!

#### "데이터가 표시되지 않아요"
- CSV 파일을 메모장으로 열어서 형식 확인
- 첫 줄이 `장소,유효폭` 형식인지 확인
- 콤마(,)가 제대로 들어갔는지 확인
